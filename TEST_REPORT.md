## 4. Сквозные сценарии (E2E)

### Сценарий 1: Пользователь без авторизации
- Открывает страницу → интерфейс на русском (по умолчанию)
- Переключает язык → все строки меняются на EN, localStorage обновлён
- Создаёт заявку → валидация на текущем языке
- Фильтрует, ищет, сортирует, меняет страницы
- Кликает на заголовок → модалка с деталями
- Меняет статус (new→in_progress→done)
- Не может изменить done (409)
- Не видит кнопку удаления

**Результат: PASS**

### Сценарий 2: Администратор
- Входит через Basic Auth → `Admin mode` / `Logout`
- Переключает на EN → `Admin mode` / `Logout`
- Видит кнопку «Удалить» в таблице и модалке
- Первый клик → «Удалить заявку #X?» с «Удалить» и «Отмена»
- Второй клик → заявка удалена (204)
- Не может удалить done (409)
- После выхода — кнопка скрыта, язык сохранён

**Результат: PASS**

### Сценарий 3: Работа с пагинацией
- 113 заявок, limit=20 → 6 страниц
- «Страница 1 из 6» / «Page 1 of 6»
- Страница 9999 → пустой массив
- Фильтр сбрасывает результаты, пагинация пересчитывается

**Результат: PASS**

### Сценарий 4: Обработка ошибок
- Пустой title → 422, сообщение на текущем языке
- Изменение done → 409, UI показывает уведомление
- Сетевые ошибки → ErrorMessage с «Повторить» / «Retry»
- SQL-инъекция → 0 результатов, без краша
- XSS-вектор → сохранён как текст

**Результат: PASS**

### Сценарий 5: Сохранение языка
- Переключить на EN → обновить страницу → интерфейс на EN
- Переключить на RU → обновить страницу → интерфейс на RU
- Язык хранится в localStorage с ключом `i18nLang`

**Результат: PASS**

### Сценарий 6: Дашборд и UX-автоматизация
- Открыть страницу → 4 карточки дашборда над формой
- Баннер зависших заявок: жёлтое предупреждение при наличии in_progress > 3 дней
- Клик по баннеру → фильтр in_progress + плавный скролл к таблице
- Сохранить фильтр: «Срочные» (priority=high, sort=created_at) → появился в выпадающем списке
- Применить сохранённый фильтр → все 5 состояний восстановлены
- Удалить пресет → исчез из списка и localStorage
- Обновить страницу → пресеты сохранены

**Результат: PASS**
